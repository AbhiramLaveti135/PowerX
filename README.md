PowerX

PowerX is a web application designed to provide users with a seamless platform for interacting with various services.  
This repository contains both the backend and frontend components of the project.

My Contribution

I was responsible for developing the frontend part of the application. My work focused on building the user interface 
and ensuring smooth interaction between the user and the backend services.

Frontend Responsibilities

- Connecting frontend components with backend APIs
- Managing client-side state and data flow
- Improving user experience and responsiveness
- Organizing frontend project structure

Features
